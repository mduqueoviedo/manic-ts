# Contributing

Thanks for helping with this project.

## Development workflow
- Create a short-lived feature branch for each change.
- Keep commits focused and descriptive.
- Open a pull request for every meaningful change.
- Use squash and merge into `main`.

## Local setup
```bash
pnpm install
pnpm dev
```

## Testing
```bash
pnpm test -- --run
pnpm build
```
