# Manic Miner - TypeScript + Vite (Canvas)

Experimental project to recreate the core logic of *Manic Miner* using HTML5 Canvas and TypeScript.

## Quick start

```bash
pnpm install
pnpm dev
```

## Tests

```bash
pnpm test
```
