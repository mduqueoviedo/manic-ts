# Manic Miner - TypeScript + Vite (Canvas)

Proyecto experimental para recrear la lógica de *Manic Miner* usando HTML5 Canvas y TypeScript.

Quick start

```bash
pnpm install
pnpm dev
```

Tests (setup básico con Vitest):

```bash
pnpm test
```

- [src/main.ts](src/main.ts)
